// The Pipeline
01 / SENDER
AI Compression
JPEG DCT + quantization shrinks image. zlib on top. 10–76x total ratio.
02 / SENDER
AES-256 Encrypt
PBKDF2 key derivation. AES-GCM with authentication tag. Wrong password = fail.
03 / SENDER
LSB Steganography
2 LSBs per audio sample. 4 possibilities per sample. <0.003% change.
04 / RECEIVER
LSB Extract
Reads modified LSBs, checks MAGIC bytes, reassembles payload.
05 / RECEIVER
AES Decrypt
Password derives key via PBKDF2. GCM tag verifies integrity.
06 / RECEIVER
AI Reconstruct
zlib → JPEG decode → bicubic upsample to original dimensions.
